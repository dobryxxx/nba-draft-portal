# NBA Draft Portal 2026

Portal editorial e dashboard de scouting para a classe do NBA Draft 2026.

## Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- DnD Kit

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Estrutura principal

- `src/views`: telas principais do produto
- `src/components`: componentes reutilizaveis
- `src/data`: base de prospects, times e validacoes
- `src/utils`: logica de apoio, como imagens e algoritmo de draft fit
- `src/styles`: tokens e helpers visuais

## Observacoes

Arquivos gerados como `node_modules`, `dist`, `.npm-cache`, logs e backups locais ficam fora do Git via `.gitignore`.
